1. ATTRIBUTE

TagName Attribute Use
h
  • style : used to add styles to an element, such as color, font, size, and more
 HTML headings are defined with the h1 to h6 tags
h1 defines the most important heading & h6 defines the least important heading
p
  • style : used to add styles to an element, such as color, font, size, and more
 HTML paragraphs are defined with the p tag
div
  • style : used to add styles to an element, such as color, font, size, and more
 The div tag defines a division or a section in an HTML document
The div tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript
The div tag is easily styled by using the class or id attribute
Any sort of content can be put inside the div tag!
img
  • src : source file
  • alt : alternative text
  • width
  • height
 HTML images are defined with the img tag
a
  • href : the link's destination
 HTML links are defined with the a tag
ul
  • style : used to add styles to an element, such as color, font, size, and more
 The ul tag defines an unordered (bulleted) list
Use the ul tag together with the li tag to create unordered lists
ol
  • style : used to add styles to an element, such as color, font, size, and more
 The ol tag defines an ordered list. An ordered list can be numerical or alphabetical
The li tag is used to define each list item
table
  • style : used to add styles to an element, such as color, font, size, and more
 The table tag defines an HTML table
An HTML table consists of one table element and one or more tr, th, and td elements
The tr element defines a table row, the th element defines a table header, and the td element defines a table cell
An HTML table may also include caption, colgroup, thead, tfoot, and tbody elements
form
  • style : used to add styles to an element, such as color, font, size, and more
 The HTML form element is used to create an HTML form for user input
The form element is a container for different types of input elements, such as: text fields, checkboxes, radio buttons, submit buttons, etc
button
  • style : used to add styles to an element, such as color, font, size, and more
 The button tag defines a clickable button
Inside a button element you can put text (and tags like i, strong, br, img, etc.). This is not possible with a button created with the input element!

2. DIFFERENTIATION

TagName Similarity Differentiation
hr & br Use to enter line breaks
  • The hr tag is most often displayed as a horizontal rule that is used to separate content (or define a change) in an HTML page
  • The br tag is an empty tag and inserts a single line break
ul & ol The tag defines a list, use together with the li tag to create each list item
  • The ul tag defines an unordered (bulleted) list
  • The ol tag defines an ordered list. An ordered list can be numerical or alphabetical
select & checkbox & radio button The input tag specifies an input field where the user can enter data
  • The select element is used to create a drop-down list
  • The checkbox element is shown as a square box that is ticked (checked) when activated
  • Radio buttons are normally presented in radio groups and only one radio button in a group can be selected at the same time
div & span The tag is used as a container for HTML elements
  • The div tag defines a division or a section in an HTML document
  • The span tag is an inline container used to mark up a part of a text, or a part of a document
script & style To link to an external document
  • The script tag is used to embed a client-side script (JavaScript)
  • The style tag is used to define style information (CSS) for a document

3. INPUT TYPE

InputType Definition
button defines a clickable button (mostly used with a JavaScript to activate a script)
checkbox defines a checkbox, the checkbox is shown as a square box that is ticked (checked) when activated, checkboxes are used to let a user select one or more options of a limited number of choices
color defines a color picker
date defines a date picker, the resulting value includes the year, month, and day
datetime-local defines a date picker, the resulting value specifies a date and time input field, with no time zone
email defines a field for an e-mail address, the input value is automatically validated to ensure it is a properly formatted e-mail address
file defines a file-select field and a "Browse" button for file uploads
hidden defines a hidden input field, a hidden field let web developers include data that cannot be seen or modified by users when a form is submitted
image defines an image as a submit button
month defines a month and year control
number defines a field for entering a number
password defines a password field (characters are masked)
radio defines a radio button, radio buttons are normally presented in radio groups, only one radio button in a group can be selected at the same time
range defines a control for entering a number whose exact value is not important (like a slider control)
reset defines a reset button which resets all form values to its initial values
search defines a text field for entering a search string
submit defines a submit button which submits all form values to a form-handler
tel defines a field for entering a telephone number
text defines a single-line text field
time defines a control for entering a time (no time zone)
url defines a field for entering a URL
week defines a week and year control (no time zone)

4. TAG DEFINITION & TESTING

4.1 strong

The strong tag is used to define text with strong importance. The content inside is typically displayed in bold.

>>> Lorem ipsum dolor sit amet consectetur, adipisicing elit. Consectetur, corrupti.

4.2 label

The label tag defines a label for several elements.

>>>





4.3 del

The del tag defines text that has been deleted from a document. Browsers will usually strike a line through deleted text.

>>> My favorite color is blue red!

4.4 b

The b tag specifies bold text without any extra importance.

>>> This is normal text - and this is bold text.

4.5 em

The em tag is used to define emphasized text. The content inside is typically displayed in italic.

>>> You have to hurry up!

4.6 mark

The mark tag defines text that should be marked or highlighted.

>>> Do not forget to buy milk today.

4.7 i

The i tag defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic.

>>> Lorem ipsum is the most popular filler text in history.

4.8 sub

The sub tag defines subscript text. Subscript text appears half a character below the normal line, and is sometimes rendered in a smaller font. Subscript text can be used for chemical formulas.

>>> This text contains subscript text.

4.9 sup

The sup tag defines superscript text. Superscript text appears half a character above the normal line, and is sometimes rendered in a smaller font. Superscript text can be used for footnotes.

>>> This text contains superscript text.
